# 🚀 FTTP Notes Generator

A modern web-based application designed to simplify FTTP troubleshooting note creation and management for support engineers.

## ✨ Features

* 📝 Generate FTTP notes quickly using predefined templates
* 🎯 Create and manage custom note templates
* 🔍 Advanced search and filtering
* 📋 One-click copy to clipboard
* 💾 Save notes locally
* ✏️ Edit and update existing notes
* 🗑️ Delete and restore notes
* ✅ Form validation for accurate data entry
* ⚡ Auto SASA details fetching
* 🌙 Dark mode support
* 📱 Fully responsive design

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Local Storage API
